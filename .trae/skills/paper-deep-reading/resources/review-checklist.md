# Paper Deep Reading 审核清单

交付前逐项检查：

## 元数据

- [ ] `name` 与 `description` 保持单篇深读定位，没有漂移成多篇综述或年度地图

## 触发边界

- [ ] 当前任务确实是单篇论文深读
- [ ] 不是多篇综述、卷次地图或子领域分布任务
- [ ] 若边界不清，已先澄清再执行

## 证据等级

- [ ] 已明确本次是 `正文级 / 摘要级 / 目录级`
- [ ] 已说明哪些判断直接来自正文
- [ ] 已说明哪些判断只是保守推断

## 输出结构

- [ ] 已完整输出 `L1-L5`
- [ ] `L2` 包含论证链，而不是一句摘要
- [ ] `L3` 每项都包含：原定义 / 常见误用 / 作者 fix
- [ ] `L4` 不是泛泛评价，而是明确指出对母学科的压力测试

## L5 硬约束

- [ ] `L5` 每条都是具体文献
- [ ] 每条都包含：`参考文献 / 角色 / 为什么关键`
- [ ] 没有用“传统 / 学派 / 路线 / 脉络”替代具体文献
- [ ] 若证据不足，已显式输出 `L5 暂缺`
- [ ] 没有把“证据不足”伪装成“安全替代版 L5”

## 失败策略

- [ ] references 不足时，已先尝试补抓参考文献区、文中点名引用或 PDF
- [ ] 若仍不足，已暂停或标明缺口，而不是自由发挥

## 质量风险

- [ ] 没有把摘要级判断写成正文级确定结论
- [ ] 没有把作者没说的内容包装成“作者观点”
- [ ] 没有把延伸背景写成正文核心论证
